# Carpeta Local Interactiva

Esta es una aplicación web local que simula una carpeta digital interactiva.

## Características
- **Diseño Premium**: Estilo "Glassmorphism" con tonos morados y toques de verde neón.
- **Interactividad**: Haz clic en la carpeta para abrirla.
- **Carga de Contenido Local**:
    - **Portada**: Puedes establecer una imagen de portada que se ve "dentro" de la carpeta cerrada.
    - **Contenido**: Agrega imágenes, videos, audios y notas de texto dentro de la carpeta.
- **Privacidad**: Todo funciona en tu navegador localmente. No se sube nada a internet.

## Instrucciones
1. Abre `index.html` en tu navegador.
2. Haz clic en la carpeta morada para abrir la vista de contenido.
3. Usa la barra de herramientas superior para agregar archivos desde tu computadora.
4. Para cambiar la portada de la carpeta, usa el botón "Portada" dentro de la vista abierta.
